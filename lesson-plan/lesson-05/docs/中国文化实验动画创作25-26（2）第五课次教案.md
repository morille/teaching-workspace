# 《中国文化实验动画创作》第五课次教案

## 一、课程基本信息

- 课程名称：中国文化实验动画创作
- 开课学期：2025-2026学年第二学期
- 授课专业：动画
- 课次：第5课次
- 学时：4学时
- 对应周次：第11周上节
- 对应章节：第二章 中国文化实验动画作品赏析
- 本次授课题目：当代创新类作品赏析与常见创作误区改进

## 二、教学内容定位

本课承接第10周下节“传统技法类作品赏析”，从水墨、剪纸、偶动画等传统技法的材料与美学经验，过渡到当代数字语境下中国文化实验动画的创新表达。课程重点由“看见传统元素”推进到“分析传统文化资源的现代转译”，引导学生理解传统文化如何在当代影像、数字绘制、合成技术、动态节奏和视听叙事中生成新的表达。

本课同时为第11周下节“前期美术设计复盘与风格规范制定”提供诊断框架。学生需要通过案例分析认识常见问题，如文化符号堆砌、风格不统一、分镜节奏与主题脱节、角色场景设定缺乏制作可行性等，并将这些问题转化为下一课次检查本组前期设计方案的依据。

## 三、教学目标（含教学重、难点）

### 1. 教学目标

#### 知识目标

1. 能够说明当代中国文化实验动画在题材、媒介、技法和传播方式上的创新特点。
2. 能够概括传统文化元素在当代动画中的视觉转译方式。
3. 能够辨析“传统符号使用”与“文化内涵转化”之间的差异。
4. 能够列举中国文化实验动画创作中常见的前期设计误区。

#### 能力目标

1. 具备从主题、风格、材料、镜头节奏和技术路径分析当代创新类作品的能力。
2. 具备发现本组项目中风格不统一、设定不完整、叙事节奏不清晰等问题的能力。
3. 具备将案例赏析结论转化为前期美术设计复盘清单的能力。

#### 素养目标

1. 在当代创新案例分析中增强文化自信，理解传统文化的创造性转化与创新性发展。
2. 在作品误区讨论中形成严谨、务实、可迭代的创作态度。
3. 在小组互评中培养尊重他人方案、基于证据表达意见的协作意识。

### 2. 教学重点与难点

- 教学重点：当代创新类中国文化实验动画的表现特征；常见创作误区及其改进思路。
- 教学难点：如何把案例分析中的审美判断转化为本组动画项目前期方案的可操作修改建议。

### 3. 学情分析

学生已经经历实验动画概述、材料语言测试、传统技法类作品赏析等学习环节，对材料、风格与中国动画学派的实验传统有初步认识。但在进入当代创新类作品分析时，学生容易出现两类偏差：一是只关注画面是否“国风”，忽略叙事结构、镜头节奏、技术路径和文化内涵之间的整体关系；二是把优秀案例当作单纯视觉参考，缺少对自身项目可行性的反向检查。

因此，本课应采用“案例赏析 + 问题归纳 + 项目诊断”的方式组织教学。教师通过当代创新类案例，引导学生从“作品为什么成立”进一步转向“自己的方案哪里还不成立”，为下一课次的前期美术设计复盘奠定方法基础。

## 四、教学方法与教学手段

- 教学方法：案例教学法、启发式教学法、讨论式教学法、项目驱动教学法、比较分析法。
- 教学手段：PPT课件、当代创新类动画案例片段、传统技法与当代创新对比图、项目诊断清单、小组讨论与课堂汇报。

## 五、教学准备

### 教师准备

1. 整理近年来中国文化实验动画与国风实验短片案例，重点选取能够体现“传统文化资源现代转译”的作品，如《中国奇谭2·耳中人》《中国奇谭2·拜山》《中国奇谭·小满》等。
2. 准备传统技法类与当代创新类作品的对比材料，突出当代作品如何从古典文本、地域民俗、材料语言、镜头节奏和技术流程等方面完成现代转译。
3. 制作“常见创作误区与改进方案”诊断表，包括主题表达、文化符号、角色设定、场景设定、色彩风格、分镜节奏、制作可行性等项目。
4. 准备课堂即时练习题和小组讨论问题。

### 学生准备

1. 回顾第10周传统技法类作品赏析内容，整理传统技法在材料、造型和审美上的特点。
2. 携带本组前期设计方案的现有材料，包括主题构思、角色草图、场景草图、色彩参考、分镜草案等。
3. 预先思考本组项目中最不确定的一个问题，准备课堂交流。

## 六、教学过程（4学时/180分钟）

### 【课前自主学习】

#### 明确目标，发布任务

教师在课前发布导学任务，要求学生完成三项准备：

1. 复习传统技法类作品赏析中关于水墨、剪纸、偶动画等内容。
2. 查找1个当代中国文化实验动画或国风实验短片案例，记录其创新点。
3. 梳理本组前期设计方案中目前最需要修改的部分。

#### 线上预习，总结归纳

学生通过线上资源完成案例查找，形成简短记录：

1. 案例使用了哪些中国文化元素。
2. 案例的风格、材料或技术路径有什么创新。
3. 案例是否存在可以借鉴到本组项目的地方。

#### 师生讨论，辅导答疑

教师根据学生预习情况进行线上答疑，重点关注学生是否只停留在图像搜集，以及是否能够分析文化转译和动画制作流程。

### 【课中讲析讨论】

#### 一、课程导入与上节内容衔接（20分钟）

教师：回顾第10周传统技法类作品赏析的核心内容，指出传统技法不仅是视觉风格，也包含材料逻辑、运动方式和民族化探索方法。

学生：结合上节案例，说出传统技法类作品最有辨识度的表现特征。

教师：提出本课问题：

1. 当代作品是否只要使用传统图案就能形成中国文化表达？
2. 数字技术进入创作后，传统文化元素可以发生哪些变化？
3. 优秀案例能帮助我们发现自己项目前期设计中的哪些问题？

设计意图：让学生从“传统技法类”的知识基础自然过渡到“当代创新类”，明确本课的案例赏析将直接服务后续项目复盘。

#### 二、第二节 当代创新类：传统文化元素的现代转译（80分钟）

教师：结合近年来的中国文化实验动画案例，讲解当代创新类作品的主要特征。本节重点转向当代创作语境，以近年作品为主要样本，分析古典文本、地域民俗、材料语言和数字制作流程如何共同参与动画表达，使传统文化元素从“外在符号”进入“主题结构、人物心理、空间行动和制作方法”。

##### 1. 当代创新类作品的总体特征

教师：讲解当代创新类中国文化实验动画的主要观察维度：

1. 文化元素从装饰符号转向叙事结构和情绪表达。
2. 传统文本、神话志怪、地方民俗不只是题材来源，也可以转化为人物心理、空间关系和主题命题。
3. 传统美学从静态图像转向动态节奏、镜头运动、声音组织和视听氛围。
4. 手工材料、数字绘制、合成技术、定格拍摄和动态设计可以形成混合媒介表达。
5. 作品的当代性体现在对现代人的情感经验、身份处境和精神困境的回应，超越单纯“国风”视觉表层。

学生：观看案例片段或案例图文材料，记录作品所使用的传统文化资源，以及这些资源如何进入主题、人物、空间、材料和制作流程。

教师：引导学生比较传统技法类与当代创新类的差异：

1. 传统技法类强调材料、工艺和民族化美术语言的探索。
2. 当代创新类强调传统文化资源在当代语境中的重新组织和主题转译。
3. 二者之间形成从传统资源、材料实验到当代叙事与数字制作方法的延展关系。

板书建议：

- 当代创新 = 传统资源 + 当代命题 + 形式实验 + 技术流程
- 观察维度：文本、民俗、材料、人物心理、空间行动、镜头节奏、制作可行性

##### 2. 知识点一：传统文本的心理化转译——以《中国奇谭2·耳中人》为例

教师：以《中国奇谭2·耳中人》为案例，分析古典志怪文本在当代动画中的心理化转译。作品取材于《聊斋志异》，创作路径从原故事复述推进到现代心理表达，将“耳中人”转化为人物内心欲望的具象形象，通过书生与耳妖之间的关系，呈现个体对真实自我的回避、冲突与和解。

讲析重点：

1. 《耳中人》突破《聊斋志异》原作叙事框架，将志怪故事转向现代心理命题。
2. “耳妖”可以理解为欲望、杂念、自我异化或内心真实声音的具象化。
3. 传统文本中的“怪”，在当代动画中可以转译成心理空间、精神困境或自我认知问题。
4. 水墨素描、留白和幽玄奇诡的视觉氛围，不只是风格选择，也服务于人物内心欲望幻境的呈现。

课堂提问：

如果小组的选题来自神话、志怪、民俗或传说，是否只能讲原故事？能不能像《耳中人》一样，把其中的怪异元素转化为人物内心的问题？

学生：结合本组选题，尝试说出传统文本或传说元素可以转化成哪一种人物心理、情绪关系或自我认知问题。

##### 3. 知识点二：民俗文化进入叙事结构——以《中国奇谭2·拜山》为例

教师：以《中国奇谭2·拜山》为案例，分析地域民俗在动画叙事中的结构作用。作品以广东河源地区和客家拜山习俗为基础，将返乡、祭祖、宗族关系和个人漂泊经验结合起来，使民俗成为推动人物行动和情感变化的核心机制。

讲析重点：

1. “拜山”从画面中的民俗符号推进为叙事结构，推动人物返乡、上山、迷路并进入幻象。
2. 客家文化、宗族关系、故土记忆和个人漂泊经验共同构成作品主题。
3. 民俗元素要和人物身份、空间行动、情绪变化发生关系，才能避免表层化。
4. 生活感与奇幻感交织，使“故土”和“先人”成为作品的情感母题，超越单纯的地方风物展示。

课堂提问：

如果作品选用了节日、仪式、地方民俗，它在作品中是一个背景，还是会推动角色行动？如果把这个民俗拿掉，故事是否还成立？

学生：对照本组项目，判断所选节日、仪式或地方民俗是否真正影响角色行动、空间路线和情绪变化。

##### 4. 知识点三：材料语言如何影响角色造型、运动和流程——以《中国奇谭·小满》为例

教师：以《中国奇谭·小满》为案例，分析剪纸定格动画中的材料语言与制作流程。作品采用剪纸与定格结合的方式，材料本身的边缘、层次、色彩和手工痕迹构成了独特的视觉风格。由于剪纸定格后期修改成本较高，前期角色造型、分层结构、动作测试和镜头设计都需要更早确定。

讲析重点：

1. 剪纸定格中的角色造型需要适应分层、摆拍、替换和拍摄等制作要求。
2. 材料边缘、纸张厚度、平面层次、手工痕迹都会成为画面风格的一部分。
3. 定格动画后期修改成本高，所以前期设计、分镜确认、动作测试必须更严谨。
4. 该案例可直接连接第10周“材质运动测试”，提醒学生把材料测试转化为项目规范，形成可持续使用的制作依据。

课堂提问：

上周做的材质运动测试如果真的进入短片制作，会影响哪些环节？角色要不要分层？背景要不要统一纸纹？动作是否需要减少复杂变形？

学生：回看本组材料语言测试或风格设想，记录材料可能影响的角色造型、背景纹理、动作复杂度和后续制作流程。

##### 5. 当代创新类案例小结

教师：总结三个案例对本课的启示：

1. 《耳中人》说明传统文本可以转化为现代心理命题。
2. 《拜山》说明民俗文化可以成为叙事结构和情感机制。
3. 《小满》说明材料语言会影响视觉风格、运动方式和制作流程。
4. 当代中国文化实验动画的重点在于传统资源进入作品主题、结构、情感和方法的深度。

学生：以小组为单位完成“传统文化元素现代转译检查表”，从文本、民俗、材料三个角度选择至少一个维度，说明本组项目可以如何深化。

设计意图：通过近年案例帮助学生理解当代中国文化实验动画的创新路径，使案例赏析直接服务于本组前期美术设计复盘和风格规范制定。

#### 三、第三节 常见创作误区与改进方案（45分钟）

教师：围绕学生项目容易出现的问题进行归纳讲解：

1. 文化符号堆砌：只把纹样、服饰、建筑作为表面装饰，缺少主题逻辑。
2. 风格参考混乱：角色、场景、道具、色彩来自不同风格体系，难以形成统一视觉基调。
3. 分镜缺少节奏：镜头只是画面排列，没有体现叙事重点、情绪递进和动作关系。
4. 设定不具备制作可行性：造型过复杂、场景层级不清、资产无法复用。
5. 材料语言与数字流程脱节：前期想法很好，但没有考虑后续分层、关键帧、合成和输出。
6. 小组分工不清：设计稿缺少责任归属，后续制作难以推进。

教师：针对每类误区给出改进方案：

1. 用一句主题表达统一文化资源选择。
2. 建立角色、场景、道具、色彩的风格规范。
3. 通过关键镜头检查叙事节奏。
4. 简化复杂造型，保留最有识别度的视觉特征。
5. 按动画制作流程提前考虑图层拆解与资产命名。
6. 明确小组成员分工和交付标准。

学生：对照本组项目，标记至少3个需要下节课重点复盘的问题。

#### 四、课堂练习与即时检测（20分钟）

教师：组织学生完成小组诊断练习。

练习任务：每组选择本组项目中的一个角色、场景或关键镜头，回答：

1. 它承载了什么中国文化意象？
2. 它的风格是否与整体项目一致？
3. 它是否具备后续制作可行性？
4. 下节课需要优先修改什么？

学生：完成简短交流，每组汇报一个问题和一个改进方向。

参考测试题：

1. 当代创新类中国文化实验动画的关键不包括哪一项？
   A. 文化内涵转化
   B. 形式语言创新
   C. 只堆砌传统纹样
   D. 技术路径服务表达
   答案：C
2. 判断前期方案是否可进入中期制作，最需要关注的是（ ）
   A. 参考图数量是否最多
   B. 角色、场景、色彩、分镜是否形成统一规范
   C. 画面是否全部复杂
   D. 是否使用最流行的软件效果
   答案：B

#### 五、课程小结与任务布置（15分钟）

教师：总结本课重点：

1. 当代创新类作品强调传统文化资源的现代转译。
2. 案例赏析应服务项目创作，推动学生形成可执行的修改依据。
3. 常见创作误区需要在前期设计复盘阶段及时修正。

学生：记录下节课需要携带和整理的材料。

## 七、课后作业与实践任务

### 作业名称

前期美术设计复盘准备清单

### 作业内容

以小组为单位整理本组前期设计方案，准备第6课次课堂复盘。需包含：

1. 主题与文化意象说明。
2. 角色设定稿或角色草图。
3. 场景设定稿或氛围参考。
4. 色彩参考与风格关键词。
5. 分镜草案或关键镜头草图。
6. 当前存在的问题清单。

### 作业要求

1. 每组至少列出3个需要修改的问题。
2. 问题应具体到角色比例、线条风格、色彩体系、场景透视、镜头节奏或制作可行性。
3. 每个问题需附带初步改进方向。
4. 下节课带入课堂进行“前期美术设计复盘与风格规范制定”。

### 作业目的

通过本课案例赏析和误区分析，引导学生把审美判断转化为项目诊断能力，为第6课次完成风格统一、设定复盘和规范制定做准备。

## 八、课程思政设计

1. 通过当代创新类案例，引导学生理解中华优秀传统文化可以在当代动画语言中持续生成新表达，是重要的创作资源。
2. 通过常见误区分析，培养学生尊重创作规律、重视制作流程、精益求精的工匠精神。
3. 通过小组互评，培养学生基于事实讨论问题、共同完善作品的团队协作意识。
4. 通过“传统技法到当代创新”的衔接，帮助学生建立文化传承与技术创新并重的创作观。

## 九、教学反思要点（授课后填写）

1. 学生是否能够区分“传统符号使用”和“文化内涵转化”。
2. 学生是否能从案例赏析中提炼可用于本组项目的改进方法。
3. 常见创作误区讲解是否足以支撑下节课的项目复盘。
4. 学生提交的复盘准备材料是否具备进入风格规范制定的基础。

## 十、配套课件建议（用于本次授课）

1. 上节回顾：传统技法类作品赏析要点
2. 本课问题：当代创新如何发生
3. 当代创新类作品特征
4. 传统文化元素的现代转译
5. 案例一：主题与视觉风格分析
6. 案例二：材料、数字技术与动态节奏分析
7. 案例三：视听语言与文化意象分析
8. 常见创作误区一：文化符号堆砌
9. 常见创作误区二：风格系统不统一
10. 常见创作误区三：分镜与制作流程脱节
11. 改进方法：主题、风格、镜头、资产四项检查
12. 小组诊断练习说明
13. 下节课复盘材料清单
