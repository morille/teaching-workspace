# 《中国文化实验动画创作》第六课次教案

## 一、课程基本信息

- 课程名称：中国文化实验动画创作
- 开课学期：2025-2026学年第二学期
- 授课专业：动画
- 课次：第6课次
- 学时：4学时
- 对应周次：第11周下节
- 对应章节：第二章 中国文化实验动画作品赏析实践环节
- 本次授课题目：前期美术设计复盘与风格规范制定

## 二、教学内容定位

本课承接第5课次“当代创新类作品赏析与常见创作误区改进”。上一课次已经帮助学生建立当代创新案例分析和创作误区诊断框架，本课将该框架转化为课堂实践，围绕各组微动画项目的前期美术设计进行复盘。

本课在动画制作流程中处于“前期方案定稿”与“中期制作准备”之间，重点解决角色、场景、道具、色彩、线条、材质、镜头风格等是否统一，以及现有设计稿是否具备后续分镜细化、资产整理、关键帧绘制和合成制作的基础。课程成果将直接衔接第12周“核心设定深度优化与资产规范化整理”，并为后续“镜头细化与资产管理”建立标准。

## 三、教学目标（含教学重、难点）

### 1. 教学目标

#### 知识目标

1. 能够说明前期美术设计复盘在动画制作流程中的作用。
2. 能够列出风格规范应包含的基本内容，如角色比例、线条规则、色彩体系、场景透视、材质表现和命名规则。
3. 能够理解前期设计与后续分镜细化、资产管理、关键帧制作之间的衔接关系。
4. 能够识别前期方案中影响中期制作的主要风险。

#### 能力目标

1. 具备对本组角色、场景、道具、色彩和分镜草案进行系统复盘的能力。
2. 具备制定基础风格规范表和问题修改清单的能力。
3. 具备根据动画制作流程判断设计稿是否可进入下一阶段的能力。
4. 具备在小组内进行任务分配、版本整理和课堂汇报的能力。

#### 素养目标

1. 在项目复盘中形成严谨规范的职业制作意识。
2. 在风格统一和资产准备中培养精益求精的工匠精神。
3. 在小组协作中增强责任分工意识和共同完成作品的团队精神。

### 2. 教学重点与难点

- 教学重点：前期美术设计复盘方法；角色、场景、色彩、镜头风格规范制定。
- 教学难点：如何把分散的创意草图整理为可进入中期制作的统一风格系统和问题修改清单。

### 3. 学情分析

学生已经完成材料语言测试、传统技法类作品赏析、当代创新类作品赏析，并开始形成中国文化实验动画微动画项目的前期构思。此阶段学生通常具有较强的视觉表达热情，但项目文件容易分散，角色与场景风格可能不统一，色彩参考缺少明确规则，分镜草案也可能只停留在故事顺序，尚未考虑镜头节奏、资产复用和后续制作可行性。

因此，本课不宜再大量新增理论内容，而应以项目实践指导为主。教师需要通过清单化、流程化的方法，帮助学生把已有创意材料整理成可检查、可修改、可分工、可继续制作的前期设计系统。

## 四、教学方法与教学手段

- 教学方法：项目驱动教学法、实践指导法、讨论式教学法、任务清单法、同伴互评法。
- 教学手段：PPT课件、前期美术设计复盘表、风格规范模板、小组项目材料、课堂展示与教师巡回指导。

## 五、教学准备

### 教师准备

1. 准备前期美术设计复盘表，包含主题表达、文化意象、角色设定、场景设定、色彩体系、分镜节奏、制作可行性等检查项。
2. 准备风格规范模板，包含角色比例、线条粗细、色彩范围、材质处理、场景透视、文件命名等内容。
3. 准备优秀前期设定案例和问题案例，用于说明规范化整理前后的差异。
4. 准备课堂指导记录表，便于记录各组需在第12周继续优化的问题。

### 学生准备

1. 携带本组前期设计方案材料，包括主题说明、角色设定、场景设定、道具设定、色彩参考、分镜草案等。
2. 根据第5课次作业，整理至少3个本组项目存在的问题和初步改进方向。
3. 明确小组成员当前分工，准备根据课堂复盘结果重新调整任务。

## 六、教学过程（4学时/180分钟）

### 【课前自主学习】

#### 明确目标，发布任务

教师发布课前任务，要求学生将本组现有前期设计材料整理为可课堂展示的文件夹或PDF，内容至少包括：

1. 项目主题与一句话故事。
2. 角色、场景、道具设计稿。
3. 色彩与风格参考。
4. 分镜草案或关键镜头草图。
5. 问题清单与修改优先级。

#### 线上预习，总结归纳

学生对照第5课次“常见创作误区”进行自查：

1. 是否存在文化符号堆砌。
2. 是否存在角色与场景风格不统一。
3. 是否存在分镜节奏不清晰。
4. 是否存在制作复杂度过高或资产无法复用。

#### 师生讨论，辅导答疑

教师在线收集各组共性问题，判断课堂中需要重点讲解的规范项，如色彩体系、角色比例、场景透视、图层拆解或命名规则。

### 【课中讲析讨论】

#### 一、课程导入与流程衔接（15分钟）

教师：回顾上一课次当代创新类案例分析和常见创作误区，指出本课任务是把“发现问题”推进为“制定规范”。

教师：说明动画制作流程中的位置：

1. 已完成：案例学习、材料测试、主题构思、前期设定初稿。
2. 本课完成：前期美术设计复盘与风格规范制定。
3. 下周进入：核心设定深度优化与资产规范化整理。
4. 后续衔接：分镜细化、素材分层、关键帧绘制、背景与贴图制作、后期合成输出。

学生：确认本组材料是否齐全，并记录本课需要完成的课堂成果。

设计意图：帮助学生理解本课不是普通修改作业，而是进入中期制作前的流程关口。

#### 二、前期美术设计复盘方法讲解（30分钟）

教师：讲解前期美术设计复盘的四个层级：

1. 主题层：故事主题、文化意象、情绪基调是否明确。
2. 视觉层：角色、场景、道具、色彩是否统一。
3. 镜头层：关键镜头是否能支撑叙事节奏和情绪变化。
4. 制作层：设计是否可拆分、可复用、可命名、可进入动画制作。

教师：说明复盘不是简单否定原方案，而是依据制作流程进行整理和优化。

学生：在复盘表上填写本组项目的主题层和视觉层自查内容。

板书建议：

- 前期复盘 = 主题清楚 + 风格统一 + 镜头可读 + 制作可行
- 本课成果 = 问题清单 + 风格规范 + 修改分工

#### 三、小组项目展示与问题诊断（45分钟）

学生：每组用3-5分钟展示本组前期设计材料，重点说明：

1. 项目主题与中国文化意象。
2. 主要角色、场景和道具。
3. 色彩与风格参考。
4. 当前最需要解决的问题。

教师：围绕每组项目进行快速诊断，重点关注：

1. 文化意象是否与故事主题相关。
2. 角色比例和造型语言是否统一。
3. 场景空间和角色运动是否匹配。
4. 色彩体系是否能支撑情绪表达。
5. 分镜是否体现关键动作和节奏变化。
6. 设计稿是否具备后续分层和资产管理条件。

学生：记录教师建议和同伴反馈，标注问题优先级。

设计意图：让学生在真实项目中理解规范要求，把抽象流程转化为具体修改动作。

#### 四、风格规范制定实践（50分钟）

教师：发放或展示风格规范模板，指导学生完成以下内容：

1. 角色规范：角色比例、头身关系、线条粗细、正侧背或关键姿态要求。
2. 场景规范：空间透视、主体区域、背景层级、可动与不可动元素区分。
3. 色彩规范：主色、辅助色、强调色、明暗范围和禁用颜色。
4. 材质规范：纸感、墨色、纹理、数字笔刷或综合材料效果的使用规则。
5. 镜头规范：关键镜头构图、景别变化、运动方向和节奏提示。
6. 文件规范：角色、场景、道具、分镜、参考资料的命名和目录结构。

学生：以小组为单位填写本组风格规范初稿，完成至少三项核心规范：

1. 角色风格规范。
2. 场景与色彩规范。
3. 分镜或关键镜头规范。

教师：巡回指导，针对不同小组提出修改建议。对于视觉风格过散的组，要求先确定主视觉参考；对于制作复杂度过高的组，要求简化角色和场景层级；对于文化表达较弱的组，要求回到主题与意象关系重新梳理。

#### 五、问题清单与修改分工（25分钟）

教师：指导学生将课堂复盘结果转化为下一阶段任务清单。

学生：每组形成“问题-修改动作-负责人-完成时间”四列表格，至少包含：

1. 角色设定修改任务。
2. 场景或道具设定修改任务。
3. 色彩体系统一任务。
4. 分镜节奏或关键镜头调整任务。
5. 文件整理与命名任务。

教师：强调第12周上节将进入“核心设定深度优化与资产规范化整理”，因此本课后的修改结果必须能够直接被整理为角色、场景、道具三类资产。

#### 六、课堂汇报、总结与任务布置（15分钟）

学生：每组用1分钟汇报本课形成的最重要修改任务和风格规范关键词。

教师：总结本课重点：

1. 前期美术设计复盘是中期制作前的必要流程。
2. 风格规范不是形式要求，而是保证团队协作和后续制作效率的工具。
3. 下节课要在本课规范基础上进行核心设定深度优化和资产规范化整理。

## 七、课后作业与实践任务

### 作业名称

前期美术设计复盘修改稿与风格规范表

### 作业内容

以小组为单位，根据课堂复盘意见完成前期设计修改，并提交一份风格规范表。内容包括：

1. 修改后的角色设定或角色比例说明。
2. 修改后的场景设定或空间层级说明。
3. 色彩体系与风格关键词。
4. 分镜或关键镜头调整说明。
5. 问题清单与成员分工表。
6. 初步工程目录结构。

### 作业要求

1. 所有修改内容需服务本组中国文化实验动画微动画项目。
2. 角色、场景、道具、色彩之间应形成统一视觉基调。
3. 文件命名应体现类别和版本，避免出现无法识别的临时文件名。
4. 每个成员需明确负责内容，为第12周核心设定优化和资产整理做准备。

### 作业目的

通过前期美术设计复盘，帮助学生完成从创意草图到规范化制作资料的过渡，确保项目能够顺利进入中期制作与资产管理阶段。

## 八、课程思政设计

1. 在项目复盘中引导学生理解中国文化表达需要扎实研究和认真转化，避免表面化、符号化使用传统元素。
2. 在风格规范制定中强调严谨、协作、守时和版本管理，培养学生的职业素养与工匠精神。
3. 在小组讨论与互评中培养学生尊重差异、共同解决问题的团队意识。
4. 在制作流程衔接中引导学生形成知行合一的学习态度，把文化自信落实到具体创作行动。

## 九、教学反思要点（授课后填写）

1. 学生是否理解前期复盘与中期制作之间的流程关系。
2. 各组是否能够形成具体、可执行的问题清单和风格规范。
3. 教师巡回指导是否覆盖到每组项目的关键问题。
4. 学生是否具备进入第12周核心设定优化与资产规范化整理的准备。
