# 中国文化实验动画创作教案生成脚本规范

## 1. 母版选择

后续课次统一使用手动调整后的第一课次文件作为母版：

`中国文化实验动画创作25-26（2）第一课次教案-沿用24-25格式 - 副本.docx`

原因：

1. 该文件已经校正封面页、页眉、表格、字号、加粗和段落密度。
2. 该文件比自动生成版更接近实际教案提交样式。
3. 后续生成时应继承母版样式，而不是重新定义整套样式。

## 2. 样式继承原则

脚本只替换文本，不主动重建表格、不重设页边距、不修改表格行高、不批量覆盖字体。

文本替换时遵守以下规则：

1. 优先保留原段落第一个有效 run 的字体、字号、加粗等样式。
2. 封面页和课程信息区继承母版的微软雅黑样式。
3. 正文表格中大多数内容继承母版宋体样式。
4. 教学内容的大板块标题继承母版的宋体 12 号加粗样式。
5. 教学设计中的教学方法标题继承母版的标题样式。
6. 不额外插入大量空段，避免破坏分页。

## 3. 内容密度规则

后续课次段落数量以手动调整版为基准：

1. 教学目标区域约 22 段。
2. 教学内容区域约 90-95 段。
3. 教学设计区域约 70-75 段。
4. 教学后记区域保留空白，不写入内容。

如果内容不足，允许保留少量空段；如果内容过多，应优先合并自然段，而不是继续新增段落。

## 4. 固定栏目结构

每一课次保持以下结构：

1. 课次、授课题目、学时。
2. 教学目标：知识目标、能力目标、素养目标。
3. 重点难点。
4. 学情分析。
5. 教学内容。
6. 教学设计。
7. 教学后记。

## 5. 教学内容写法

教学内容区沿用以下板块：

1. 【课前自主学习】
2. 明确目标，发布任务
3. 线上预习，总结归纳
4. 师生讨论，辅导答疑
5. 【课中讲析讨论】
6. 一、课程内容分享/上节实践作业展示
7. 二、教师讲评，评价学习效果
8. 三、教师精讲，引导合作探究
9. 四、课终小结
10. 五、作业布置
11. 【课后实践应用】

课堂测试应写成可以直接使用的完整题目，优先保留选项和答案。

## 6. 教学设计写法

教学设计区保留母版空行节奏，重点写清楚：

1. 课前启发式教学。
2. 课中 180 分钟分配。
3. 教师活动。
4. 学生活动。
5. 课程思政设计。
6. 内容总结。
7. 作业布置。

`教师：`、`学生：` 前缀应保留，并继承母版中前缀强调样式。

## 7. 课次内容对齐规则

生成新课次前必须先对齐新版教学进度表。

第2课次对应内容：

1. 1.2 实验动画的分类。
2. 按材料：综合材料的应用。
3. 剪纸动画、实物动画、偶动画。
4. 1.3 中国文化实验动画的探索。

因此第2课次不应回到旧版“实验动画制作原理与方法”的完整内容，而应围绕材料分类和中国文化实验动画探索展开。
